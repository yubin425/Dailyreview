나만의 메인 화면

어떤 기록을 남겨왔나요? 당신만을 위한 기록들을 보여줄게요.
보았던 영화들을 기반으로 포스터를 보여주고, 통계를 산출합니다.
바로 영화 검색으로 이동할 수도 있어요!

원하는 영화는 모두 이곳에!

KMDB API 기반의 검색 기능으로 원하는 영화를 간편하게 찾고
리뷰를 남기고, 위시리스트에 등록해보세요!
추후 정확도와 인기순 등 검색 기능을 개선할 예정이랍니다.

간편한 기록 & 커스터마이징

같이 본 친구도 쓰고 싶고, 몇회차인지도 적고 싶고, OTT도 적고 싶고...
개인마다 적고 싶은 항목은 다른데, 무조건 모두 적어야 하거나 적을 수 없어서 불편했을거에요.
데일리뷰에서 원하는 리뷰 레이아웃을 직접 만들어 저장해보세요!
가볍게 적을 수도 있고, 꽉꽉 채워 적을 수도 있답니다. 당신이 원하는 대로 사용해보세요.
이후 영화 이외에도 적을 수 있는 다양한 기본 레이아웃과 커스텀, 추천 기능을 추가할 예정이에요!

인증샷도 찰칵! 친구들과도 공유하는 나의 일상.

리뷰들을 쉽게 조회하고, 수정하고, 친구와 공유해보세요.
원하는 대로 인증샷 이미지를 꾸미고 저장할 수 있어요!
다양한 꾸미기 레이아웃(무비 티켓, 영수증, 통계 출력)을 제공할 예정이에요!

위시리스트를 만들고 공유하세요.

음악에는 플레이리스트 공유가 있는데, 다른 매체에는 왜 없을까요?
데일리뷰에서 위시리스트를 작성하고 친구와 공유해보세요.
같이 도장깨기를 하거나, 취향을 공유할 때 유용하겠죠!
